# PHP para iniciantes


###### echo
```php
<?php
    echo "oi";
?>
```


###### Variaveis
```php
<?php

$meuNome = "Fulano";

echo $meuNome; 

// o resultado será :  Fulano

?>
```

###### Interpolação
```php
<?php

$meuNome = "Fulano";

echo "oi $meuNome, como vai?"; 

// o resultado será :  oi Fulano, como vai?

?>
```
