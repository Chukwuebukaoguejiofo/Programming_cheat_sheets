# NGINX

###### https://www.sylvaindurand.org/setting-up-a-nginx-web-server-on-macos/

```bash
# terminal

# Install nginx
brew install nginx 

# Edit configuration
subl /usr/local/etc/nginx/nginx.conf

# Start nginx
sudo nginx

# Restart Nginx to activate the changes:
sudo nginx -s reload

# http://localhost:80
```
