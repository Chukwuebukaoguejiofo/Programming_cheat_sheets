### Download images from docker hub
```bash
$ docker pull <image>
```

### List downloaded images
```bash
$ docker images
```

### Boot and exit an image
```bash
$ docker run <image> <command>
```

### Show running containers
```bash
$ docker ps
$ docker ps -a # show all
```

### Start a shell session
```bash
$ docker run -it <image> sh
```
